# لعبتي ثلاثية الأبعاد 🎮

لعبة 3D بسيطة مبنية باستخدام Three.js، مخصصة لمنصة Xogot.

## المميزات
- شخصية ثلاثية الأبعاد تتحرك في عالم مفتوح
- جمع المكعبات الذهبية لزيادة النقاط
- دعم التحكم باللمس للجوال
- أشجار وبيئة طبيعية
- إضاءة وظلال متحركة

## التشغيل
افتح `index.html` في المتصفح أو استورد المشروع إلى Xogot.

## التحكم
- **لوحة المفاتيح:** الأسهم أو WASD
- **الجوال:** اللمس والسحب
